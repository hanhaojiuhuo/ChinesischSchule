# 海尔布隆一心中文学校 – Yi Xin Chinesische Sprachschule Heilbronn

Website for the Yi Xin Chinese Language School in Heilbronn, built with [Next.js](https://nextjs.org), [Tailwind CSS v4](https://tailwindcss.com), and TypeScript.

The visual style follows the school's circular badge logo: deep red (`#c41230`), dark (`#1a1a1a`), and white color palette with Chinese-compatible system font stacks.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.
